*(5Б) Какими способами можно задать значения для данных-членов структуры?

можно в main/функции или можно при создании при помощи: 
list = {data1,data2,...};
list.data_1 = data1;

*(5Б) Конструкторы и деструкторы в структурах.

деструкторы выполняется при выходе структуры из области видимости
конструкторы выполняются при создании структуры

*(5Б) Отличие структур и классов в  С++, для чего следует использовать структуры.

В классах по умолчанию стоит private, а в структурах public, а так они похожи. 
Структуры можно использовать для описания нескольких объектов, включающие одинаковые поля

*(5Б) Конструкторы и деструкторы, для чего нужны.

деструкторы используются для удаления структуры перед выходом из области видимости/завершении программы;
можно передать хранящиеся в ней данные в какие-то данные, чтобы ничего не потерять

конструкторы: первоначальная обработка начильних данных, их принт;

*(5Б) Перечисления (enum), и перечисления с областью видимости (enum class). Для чего нужны, особенности перечислений с областью видимости.

На этапе предпроцессора на место переменных перечисления вставляются их значения(как define); 
Для enum class abcd{data1,...} в main нужно писать abcd::data1
